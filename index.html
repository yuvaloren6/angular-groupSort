A bootstrap based alternative angular select with ng-model support


The dropdownSelect directive is an attribute only directive.

Proerties of the directive:

title: The title of the select input control (will also appear as first list element).
title-class: Since the control is based on a bootstrap dropdown, you must provide a bootstrap button class such as "btn-default", "btn-success" etc.
options: The array of objects whose proprties you wish to include in the select.
caption: The property name whose values are presented to the user in the select elements list.
value: The property name whose value is set in the ng-model when the user select a list element.
group-by (optional): The property by used to group list elements into different groups. When the select control opens, the value of the group-by-property is listed as an-unselectable list item (in a special css class) before any of the list items in the group

providing thermal response to the user:
selectedClass (optional): A hidden property that is looked up in the object representing a selection. if such a property is found when the object is selected by the user, then the value of this property will replace the value of title-class. This enable you to let the user know that this option is valid, but not recommended. The second control provide such an example.

The following control shows a dropdown select whose list elements, countries, are grouped in continents. When you make a selection, the continent of the selected country is presented on the right as it is assgined by the ng-model

The dropdownSelect directive

<div dropdown-select class="btn-group" ng-model="continent" title="Country" title-class="btn-default" options="countries" caption="country" value="continent" group-by="continent"> </div>

Value (two way binding) of dropdownSelect is displayed through ng-bind

<p>continent: <i class="color-enhance">{{continent}}</i></p>
